### hexo使用说明
1. git bash中使用git pull，确保本地与github最新文件同步
2. hexo clean
3. hexo new post 标题 ,创建新文章
4. 编辑文章
5. hexo s，hexo d, hexo clean
6. git add. , git commit -m "说明" , git push
